* JSP(Java Server Page)

JSP란, HTML형식 내에서 자바코드를 기술할 수 있는 "자바 언어"
서블릿(JAVA코드내)에서 응답화면(HTML)을 구현했떤 복잡함을 보다 간단하게 해결 가능
=> 즉, JSP의 가장 큰 장점은 Servlet에서는 사용자의 요청 처리(비즈니스 로직처리)에만 집중
응답화면을 만들어내는 건(프레젠테이션 로직) JSP에서 집중하게끔 분리할 수 있는 것

JSP Elements 표현법(JSP요소 표기법)

1. JSP 스크립팅 원소(Scripting Element)

JSP 페이지에서 자바 코드를 기술 할 수 있게 해주는 기능

  1. 선언문 : < %! 자바코드 % >
    멤버변수와 메소드를 정의해놓기 위해 사용

  2. 스클림틀릿 : < % 자바코드 >
    JSP에서 자바코드 기술하기 위한 기본적인 표현법

  3. 표현식(출력식) : < %= 자바코드 %>
    자바에서 작성한 값(또는 메소드 호출)을 화면상에 출력하기 위해 사용

2. 지시어 (Directive) <%@ page|include|taglib 속성명="속성값" 속성명="속성값" %>

JSP page 전체에 영향을 미치는 정보를 기술 할 때 쓰임

  1. page 지시자 : 현재 jsp페이지를 처리한데 필요한 각종 속성들을 기술하는 부분(jsp문서의 맨앞(상단)에 위치)
    • language : 사용할 스크립트 언어 유형을 지정하는 속성
    • contentType : 페이지의 형식, 인코딩 방식을 지정하는 속성
    • pageEncoding : JSP파일에 기록된 자바코드의 인코딩 방식을 지정
    • import : 자바의 import와 같은 의미
